# Cocktail Roulette 

A fun and interactive web application that generates random cocktail recipes. Discover new drink recipes every time you refresh the page!

## Description

Cocktail Roulette is a web application built with Node.js, Express, and EJS that fetches random cocktail recipes from TheCocktailDB API. Each time you visit the page, you'll get a new random cocktail with its ingredients, instructions, and an image.


Refresh the page to get a new random cocktail recipe!


## API

This project uses [TheCocktailDB API](https://www.thecocktaildb.com/api.php) to fetch random cocktail data.
